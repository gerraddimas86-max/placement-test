# English Placement Test

Aplikasi placement test bahasa Inggris berbasis web untuk mengukur kemampuan grammar dan vocabulary.

## 🚀 Fitur

- 5 soal placement test (grammar & vocabulary)
- Submit jawaban dengan processing async
- Polling otomatis untuk cek hasil
- Feedback level bahasa Inggris (Beginner - Advanced)

## 🛠️ Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **HTTP Client:** Axios
- **Deployment:** Vercel

## 📦 Installation

```bash
# Clone repository
git clone https://github.com/USERNAME/placement-test.git

# Masuk ke folder
cd placement-test

# Install dependencies
npm install

# Buat file .env.local (copy dari .env.example)
cp .env.example .env.local

# Jalankan development server
npm run dev
